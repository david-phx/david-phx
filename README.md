I'm a software developer & competitive shooter. When I'm not writing code for
robots at [Handwrytten](https://www.handwrytten.com), I build tools for
shooting sports. Most of the projects started as tools for my own practice and
matches — now other competitors use them too, including world class shooters.

Everything competitive shooting related lives at
[Gunz N' Coffee](https://gunzncoffee.com/).

## Projects

[**Steel Challenge Score Tracker**](https://gunzncoffee.com/scoretracker/)

[App Store](https://apps.apple.com/app/steel-challenge-score-tracker/id6744840168) ·
[Google Play](https://play.google.com/store/apps/details?id=com.gunzncoffee.scoretracker)

A complete companion for Steel Challenge competitors. Track match history for
any shooter across every division and stage, analyze performance and progress
toward the next classification, and see how shooters rank against each other.

*JavaScript · React Native · Expo · Python · AWS Lambda · DynamoDB*

[**Steel Challenge Stages**](https://steelchallengestages.com/)

[steelchallengestages.com](https://steelchallengestages.com/)

Renders Steel Challenge stages from a first-person view and prints them for
practice. Scales targets to the chosen shooting box, eye level, and distance to
the wall for at-home dry-fire, or prints full stage diagrams for building the
real layout on a range. Includes a stage designer for laying out custom stages
from scratch.

*TypeScript · React · Next.js · Tailwind CSS*
<br clear="left">

<img src="assets/partimer.png" width="64" align="left">

[**Par Timer**](https://gunzncoffee.com/partimer/)

[App Store](https://apps.apple.com/app/par-timer-dry-fire-trainer/id6784820515) ·
[Google Play](https://play.google.com/store/apps/details?id=com.gunzncoffee.partimer)

A configurable par timer for dry-fire practice. Delivers timing cues by display,
audio, and vibration, with adjustable start delays, par times, shot cues, spoken
voice prompts, and how par times tighten across sets. Records a history of every
run to track training consistency over time.

*JavaScript · React Native · Expo*
<br clear="left">

<img src="assets/targets.png" width="64" align="left">

[**Customizable Zeroing Targets**](https://gunzncoffee.com/targets/)

[gunzncoffee.com/targets](https://gunzncoffee.com/targets/)

Generates free printable zeroing targets. Takes a zero distance, an available
shooting distance, caliber, and sight height over bore, then computes the
ballistic trajectory to plot where shots should land for a correct zero and
lays it out as a calibrated target printed at 100% scale — letting shooters
confirm a zero from whatever distance their range offers.

*JavaScript*
